With bases larger than ten, we need different symbols for the numbers called ten, eleven, twelve, thirteen, fourteen, etc., in base ten. Here is how they are represented:

  • What is called ten in base ten will be represented with the digit T.
  • What is called eleven in base ten will be represented with the digit E.
  • What is called twelve in base ten will be represented with the digit D.
  • What is called thirteen in base ten will be represented with the digit R.
  • What is called fourteen in base ten will be represented with the digit F.
